Subject of a Treasure Hunt

‘The subject of a treasure hunt’ is a mobile app designed to help new students learn about the facilities and resources provided by the university.

Through the use of user location and mapping, the app aims to serve students with riddles relevant to their degree, which can then be solved by exploring specific locations containing the answer. Utilizing an achievement based reward system, participants are motivated to answer more riddles by exploring each location thoroughly.

By bringing an objective to campus orientation, riddles can prompt students to explore the campus and learn important information in the process. Our implementation of riddles in a location-aware context would aim to provide students with an app they can use on campus in their own time, with potential for the university to expand later down the road.

This application was built in 2020 as a Group Project for David Bainbridge as part of COMPX241 "Smoke and Mirrors".

Source code can be found my GitHub.

HortPlus SprayPlan Manager MVP

As part of a Software Engineering industry project, the objective was to rebuild the legacy Spray Plan system with a modern tech stack. I worked in a team of 4 to develop a MVP for HortPlus.

Bhavit and I built the entire frontend from scratch Using React TypeScript, React Router, Formik and Bootstrap. The result was a successful MVP product which launched shortly after the project's conclusion.
